# Assets

This folder stores visual source material and implementation-ready exports.

## Current Assets

- `Screens/product-direction.png`: supplied product direction mockup.

## Planned Structure

- `Icon/`: layered source files, icon studies, and exported app icon assets.
- `Screens/`: product mockups and implementation screenshots.

## Asset Rules

- Keep source files editable when possible.
- Keep generated exports separate from source artwork.
- Do not use vendor logos as the primary app identity.
- Prefer neutral operating-system metaphors for wInstaller branding.
- Track license and provenance for any third-party asset.
