# App Icon Assets

Final icon source files belong here.

Recommended files:

- `winstaller-icon.icon`: Icon Composer source, if used.
- `winstaller-icon.fig`: Figma source, if used.
- `winstaller-icon.sketch`: Sketch source, if used.
- `AppIcon.appiconset`: Xcode export.

The icon direction is defined in [../../ICON_GUIDELINES.md](../../ICON_GUIDELINES.md).
